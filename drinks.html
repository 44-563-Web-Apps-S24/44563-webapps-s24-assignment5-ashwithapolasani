The drink names row (with class row) will break first as the browser window shrinks. This is because (div) elements with the class col-md—which defines a column width for medium-sized screens—contain the drink names. These columns will stack vertically when the screen size decreases, breaking the row. However, because the Bootstrap col-sm-2 class is used to specify column width for small-sized displays, the letters row (with class row and col-sm-2) will adjust to smaller screen sizes more effectively.
